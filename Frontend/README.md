@Features@

Landing Page: 

Shows two buttons & Create New Pool and Join Existing Pool.

Create Pool:

Input fields: Pool Topic, Votes per Participant, increment/decrement buttons, Enter Name.

Buttons: Create and Start Over.

Pool ID:

After pool creation, the app displays the pool topic and generates a unique Pool ID.

Digits are shown in orange, alphabets in blue.

The Pool ID can be copied and shared with others to join the pool.

Joining Pool:

When users join a pool via Pool ID, the creator sees updated joined icons for each participant.

A pen icon next to participants opens the nomination page.

Nominations:

Users can submit their nominations in an input field.

All participants can see others’ nominations in real time.

Pool creator can delete any participant’s nominations one by one; other participants can only modify their own nominations.

The pen icon also displays the total number of nominations.

Voting:

After nominations are complete, the pool creator starts the voting phase.

Participants vote according to their preferences and submit votes.

Results:

After all votes are submitted, the pool creator ends the voting phase.

Results are displayed to all participants in real time.
